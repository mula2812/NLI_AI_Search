Windows

  1. הורד את קובץ ה-ZIP
  2. חלץ את הקבצים
  3. הרץ את setup.exe

  1. הורד את קובץ ה-tar.gz
  2. חלץ את הקבצים
  3. הרץ את ./install.sh
